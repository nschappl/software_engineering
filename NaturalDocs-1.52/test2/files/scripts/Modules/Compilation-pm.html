sub Exec_GetParam #($cas)

Build the list of compilation option in fonction of the name of the library and of the machine option, defined in Modules/Machines/Machine.pm.

Here is the list of all keywords

_nompiAdd the flad -DFORCE_NOMPI
_mpiNothing, opposite to _nompi
_nosmpAdd the flag -DFORCE_NOSMP
_smpNothing, opposite to _nosmp
_staticNothing, opposite to _dyn
_dynAdd the flag -DPASTIX_DYNSCHED
_intUses default integer type
_longUses long integers
_int32Uses 32 bits integers
_int64Uses 64 bits integers
_simpleUses simple float precision
_doubleUses double float precision
_realUses real float
_complexUses complex float
_scotchUses scotch
_metisUses metis
_distUses distributed interface
_consoUses -DFORCE_CONSO
